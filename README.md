This script continuously checks whether a user-provided website is online by sending HTTP requests in an infinite loop. It reports the website’s status code, uptime, and errors directly in the terminal. The loop keeps running until stopped manually, allowing you to monitor stability in real time.
Perfect for beginners exploring network automation, DevOps, or Python scripting.

⸻

Features
	•	Infinite uptime monitoring loop
	•	User enters any website URL
	•	Color-coded online/offline messages
	•	Automatic retries
	•	Lightweight and beginner-friendly
	•	Stops instantly with CTRL + C
